# Python scripts for geometry formuls

In this project we represent you some scripsts for calculating perimeter and area of such geometric figures like triangle, rectangle, square and circle.

## Functions and examples

There are examples of usage for each function

### Triangle - area(a, h):

#### Parameters: 
a - foundation
b - high

#### Result:

Area of triangle with each high and foundation

#### Example:

```
area(5, 2) # 5
``` 

### Triangle - perimeter(a, b, c):

#### Parameters: 
a, b, c - modular sizes in decimal numbers of each sides

#### Result:

Perimeter of triangle with each sides.

#### Example:

```
perimeter(5, 2, 3) # 10
```

### Rectangle - area(a, b):

#### Parameters: 
a, b - sides that are not in front of each other.

#### Result:

Area of rectangle with such sides.

#### Example:

```
area(5, 2) # 10
```
### Rectangle - perimeter(a, b):

#### Parameters: 
a, b - sides that are not in front of each other.

#### Result:

Perimeter of rectangle with such sides.

#### Example:

```
perimeter(5, 2) # 20
```

### Square - area(a):

#### Parameters: 
a - side

#### Result:

Area of square with such side.

#### Example:

```
area(5) # 25
```

### Square - perimeter(a):

#### Parameters: 
a - side

#### Result:

Perimeter of square with such side.

#### Example:

```
perimeter(5) # 20
```

### Circle - area(r):

#### Parameters: 
a - radius

#### Result:

Area of circle with such radius.

#### Example:

```
area(5) # 25 * pi
```

### Circle - perimeter(r):

#### Parameters: 
a - radius

#### Result:

Perimeter of circle with such radius.

#### Example:

```
area(5) # 10 * pi
```

# Math formulas that were used
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = h * a / 2

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c
